# Tutorials

The tutorial notebooks in this directory are stripped of their outputs
to keep the repository lightweight.  **Run them yourself** to see the
results – they are fully functional.

## How to run a tutorial

1. Install the library with optional dependencies:
   ```bash
   pip install metaheuristic-designer[examples]
   ```

2. Start Jupyter and open any notebook:
   ```bash
   jupyter notebook tutorials/genetic_algorithm_tutorial.ipynb
   ```
   or use VS Code / JupyterLab.

3. Execute all cells (`Cell → Run All`).  The outputs (plots, statistics,
   convergence curves) will appear as the tutorial runs.

All tutorials are self‑contained and can be executed in any order.
